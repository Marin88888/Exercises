# Exercises

## Next steps after your first push

If this was your first push, here are common next steps:

1. Check the repository on GitHub to confirm your commits are visible.
2. Review open issues (or create one) to describe what you plan to work on next.
3. Create a new branch for your next change, then open a pull request when it is ready.
4. Update this README with any project details you learn as you go.
